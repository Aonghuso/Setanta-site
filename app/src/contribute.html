Setanta needs help!

Looking for fluent Irish speakers interested in helping make learning programming available through Irish.

Setanta was created to enable people to learn to program through Irish, that is the primary design goal. The last obstacle in the way of this goal is making the tutorial fully available in Irish and that is where I need help.

A lot of time was spent on creating a great online interactive tutorial system for Setanta, but unfortunately that tutorial is only currently available in English. Setanta's goal to make Irish programming accessible can't be complete until the tutorial is available in Irish. My Irish is not up to scratch to do it entirely myself. This is why I need help from fluent speakers to proof read and correct my faulty Irish. I want this tutorial to be done well, the project can't hope to make learning programming through Irish possible if the Irish on the site is broken. By sending out this message I hope I can find a fluent Irish speaker who can help make sure that the Irish in Setanta is the best it can be.

Any help at all would be really greatly appreciated, anything from answering ad-hoc questions about phrasing, to full involvement in translation would go a long way toward making Setanta fully achieve the goal of making learning programming in Irish possible.

Please take a look at the current English tutorials here, and if you're interested in helping out, feel free to send me an email at eoin@try-setanta.ie or contact me on Twitter.

Míle Buíochas!

NB: Setanta is free, open source and volunteer developed.
